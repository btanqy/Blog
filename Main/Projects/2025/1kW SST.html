1kW Solid State Transformer

The motivation behind making this is to be able to have a modular supply to get high voltage AC and low voltage DC to test the other supplies and projects, or more specifically the farnsworth fusor and Class E plasma generator. Having swappable modules that can output and vary controlled voltage and current will be more helpful than using a generic variac or DC supply because of its high, but variable output. This project uses a dual active bridge for the topology. There are four sections of this project: PFC (power factor correction) design, SST (solid state transformer) Design, Control, and Implementation.
